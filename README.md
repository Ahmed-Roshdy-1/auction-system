# Auction System

A minimal auction system written on Java. A college assignment.

## Dependencies

Required:

- Java 17.
- Gradle 7.

Network retrieved:

- OpenJFX.
- Gson.

## Build and Run

Start the server. Uses `server.port` in `gradle.properties` and defaults to 8000.

```
gradle server:run
```

Run the client application.

```
gradle client:run
```
